Slack

Slack é uma plataforma de comunicação empresarial que organiza conversas em canais, substituindo e-mails internos e promovendo uma comunicação mais rápida e contextualizada. É amplamente utilizado por equipes de todos os tamanhos para colaboração e coordenação de projetos.

Principais Funcionalidades:

  • Canais públicos e privados para organizar conversas por projeto, equipe ou tópico.
  • Mensagens diretas para conversas individuais ou em pequenos grupos.
  • Threads para manter discussões focadas dentro de um canal.
  • Compartilhamento de arquivos e integrações com centenas de outros aplicativos (Google Drive, Asana, GitHub, etc.).
  • Chamadas de áudio e vídeo.

Mini Tutorial de Como Utilizar:

  1. Entrar em um Workspace: Geralmente, você receberá um convite por e-mail para se juntar a um workspace do Slack da sua empresa ou equipe. Siga as instruções para criar sua conta.
  2. Navegar pelos Canais: Na barra lateral esquerda, você verá a lista de canais (#geral, #projetox, etc.). Clique em um canal para ver as mensagens. Participe de canais relevantes para suas atividades.
  3. Enviar Mensagens: Digite sua mensagem na caixa de texto na parte inferior do canal ou da mensagem direta e pressione Enter. Use "@nome_do_usuário" para mencionar alguém.
  4. Usar Threads: Para responder a uma mensagem específica sem poluir o canal principal, passe o mouse sobre a mensagem e clique no ícone "Responder na thread".
  5. Configurar Notificações: Ajuste suas preferências de notificação para evitar sobrecarga, recebendo alertas apenas para menções diretas ou atividades importantes.

Zoom

Zoom é uma plataforma de videoconferência e colaboração online que se tornou essencial para reuniões virtuais, webinars e eventos online. Oferece vídeo e áudio de alta qualidade, compartilhamento de tela e recursos interativos.

Principais Funcionalidades:

  • Reuniões com vídeo e áudio em HD.
  • Compartilhamento de tela (desktop, aplicativo específico, quadro branco).
  • Gravação de reuniões (local ou na nuvem).
  • Chat durante as reuniões.
  • Salas de breakout para dividir participantes em grupos menores.
  • Webinars para grandes audiências.

Mini Tutorial de Como Utilizar:

  1. Instalar o Aplicativo: Baixe o cliente Zoom em zoom.us/download para desktop ou o app para dispositivos móveis.
  2. Entrar em uma Reunião: Clique no link da reunião fornecido pelo anfitrião ou abra o aplicativo Zoom, clique em "Ingressar em uma Reunião" e insira o ID da Reunião.
  3. Controles Básicos: Durante a reunião, você encontrará controles na parte inferior para ativar/desativar seu microfone (Mudo) e vídeo (Iniciar/Parar Vídeo), ver participantes, abrir o chat e compartilhar tela (se permitido).
  4. Agendar uma Reunião (se for anfitrião): No aplicativo, clique em "Agendar". Configure a data, hora, opções de segurança (como senha e sala de espera) e outras preferências. Depois, compartilhe o link do convite.

Asana / Trello (Gerenciamento de Projetos)

Asana e Trello são duas das mais populares ferramentas de gerenciamento de projetos e tarefas, ajudando equipes a organizar, rastrear e gerenciar seu trabalho. Ambas oferecem abordagens visuais para o fluxo de trabalho.

Asana:

Focado em listas de tarefas e projetos, com cronogramas, quadros e portfólios para gerenciar iniciativas complexas.

  • Criação de projetos, tarefas e subtarefas.
  • Atribuição de responsáveis e prazos.
  • Visualizações de lista, quadro (Kanban), calendário e cronograma (Gantt).
  • Automação de fluxos de trabalho.

Visite o site oficial do Asana

Trello:

Utiliza um sistema de quadros Kanban com cartões e listas, ideal para visualizar o progresso de tarefas de forma simples e intuitiva.

  • Quadros para projetos ou fluxos de trabalho.
  • Listas para representar etapas (Ex: A Fazer, Fazendo, Feito).
  • Cartões para tarefas individuais, que podem conter checklists, anexos, comentários.
  • Power-Ups para adicionar funcionalidades extras.

Visite o site oficial do Trello

Mini Tutorial de Como Utilizar (Conceito Geral para Trello/Kanban):

  1. Criar um Quadro/Projeto: Dê um nome ao seu projeto (ex: "Lançamento Novo Produto").
  2. Definir Listas/Etapas: Crie colunas que representem o fluxo de trabalho (ex: "Ideias", "A Fazer", "Em Andamento", "Revisão", "Concluído").
  3. Adicionar Cartões/Tarefas: Para cada tarefa, crie um cartão na lista apropriada (ex: "Criar protótipo" na lista "A Fazer"). Adicione detalhes ao cartão: descrição, responsável, prazo.
  4. Mover Cartões: À medida que o trabalho progride, arraste os cartões entre as listas (ex: mova "Criar protótipo" de "A Fazer" para "Em Andamento" quando iniciar, e depois para "Concluído").

VSCode / GitHub (Desenvolvimento de Software)

Para desenvolvedores, o Visual Studio Code (VSCode) é um editor de código-fonte leve, mas poderoso, e o GitHub é uma plataforma de hospedagem de código e colaboração usando o sistema de controle de versão Git.

Visual Studio Code (VSCode):

Um editor de código gratuito, altamente personalizável, com suporte para inúmeras linguagens, depuração integrada, controle Git e um vasto ecossistema de extensões.

  • Edição de código inteligente com IntelliSense (autocompletar).
  • Depurador integrado.
  • Terminal integrado.
  • Suporte nativo a Git.
  • Marketplace de extensões para adicionar funcionalidades (linters, formatadores, temas).

Visite o site oficial do VSCode

GitHub:

Plataforma líder para versionamento de código com Git, permitindo que desenvolvedores armazenem seus projetos, colaborem com outros, rastreiem alterações e gerenciem o ciclo de vida do desenvolvimento de software.

  • Hospedagem de repositórios Git (públicos e privados).
  • Ferramentas de code review (Pull Requests).
  • Issue tracking para gerenciamento de bugs e tarefas.
  • GitHub Actions para automação de CI/CD.
  • Wikis e páginas de projeto (GitHub Pages).

Visite o site oficial do GitHub

Mini Tutorial de Como Utilizar (Fluxo Básico VSCode + GitHub):

  1. Instalar VSCode e Git: Baixe e instale o VSCode. Baixe e instale o Git (git-scm.com).
  2. Criar/Clonar Repositório: No GitHub, crie um novo repositório. No seu computador, use o terminal (ou o terminal integrado do VSCode) para clonar o repositório: `git clone [URL_DO_REPOSITORIO]`. Abra a pasta do projeto no VSCode.
  3. Fazer Alterações: Crie ou modifique arquivos no VSCode. Salve as alterações.
  4. Versionar Alterações (Commit): No VSCode (aba Controle do Código Fonte) ou no terminal:
    • Adicione os arquivos modificados ao "stage": `git add .` (para todos) ou `git add nome_do_arquivo`.
    • Faça o commit das alterações com uma mensagem descritiva: `git commit -m "Minha primeira alteração"`.
  5. Enviar para o GitHub (Push): Envie seus commits para o repositório remoto no GitHub: `git push origin main` (ou o nome da sua branch principal).
